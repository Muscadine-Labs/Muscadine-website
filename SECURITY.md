# Security Policy

## Reporting a Vulnerability

If you believe you have discovered a security vulnerability in any Muscadine product, service, or infrastructure, please report it to us responsibly.

**Contact:** [muscadinelabs@gmail.com](mailto:muscadinelabs@gmail.com)

Please include:

- A clear description of the issue
- Steps to reproduce (if applicable)
- Impact assessment, if possible
- Your contact information so we can follow up

We will acknowledge receipt and aim to respond with next steps within a reasonable timeframe. We ask that you do not disclose the issue publicly until we have had a chance to address it.

Thank you for helping keep Muscadine and our users secure.
